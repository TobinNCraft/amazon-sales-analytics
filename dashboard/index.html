Common Table Expressions (CTEs)

Used for complex multi-step analysis including RFM customer segmentation, cohort analysis, and hierarchical revenue calculations.

WITH monthly_revenue AS (
    SELECT DATE_TRUNC('month', order_date) AS month,
           SUM(revenue_usd) AS total_revenue
    FROM orders
    GROUP BY 1
),
revenue_with_metrics AS (
    SELECT month, total_revenue,
           AVG(total_revenue) OVER (
               ORDER BY month 
               ROWS BETWEEN 2 PRECEDING AND CURRENT ROW
           ) AS revenue_3m_avg
    FROM monthly_revenue
)

Window Functions

Applied for running totals, moving averages, ranking, percentiles, and growth rate calculations across time series data.

SELECT category, total_revenue,
    RANK() OVER (ORDER BY total_revenue DESC) AS rank,
    ROUND(
        SUM(total_revenue) OVER (
            ORDER BY total_revenue DESC
        ) * 100.0 / SUM(total_revenue) OVER (), 2
    ) AS cumulative_pct,
    NTILE(5) OVER (ORDER BY total_revenue) AS quintile
FROM category_metrics

Advanced Analytics

Implemented RFM Analysis, ABC Classification, Pareto (80/20) Analysis, and Cohort Retention tracking for actionable business insights.

-- RFM Customer Segmentation
SELECT customer_id,
    NTILE(5) OVER (ORDER BY recency DESC) AS r_score,
    NTILE(5) OVER (ORDER BY frequency) AS f_score,
    NTILE(5) OVER (ORDER BY monetary) AS m_score,
    CASE 
        WHEN r >= 4 AND f >= 4 AND m >= 4 THEN 'Champions'
        WHEN r >= 4 AND f >= 3 THEN 'Loyal Customers'
        ELSE 'Needs Attention'
    END AS segment

Database Design

Designed a normalized star schema with dimension and fact tables, proper indexing strategies, and optimized views for analytics.

-- Star Schema Design
CREATE TABLE orders (
    order_id VARCHAR(20) PRIMARY KEY,
    customer_id INT REFERENCES customers(id),
    region_id INT REFERENCES regions(id),
    order_date DATE NOT NULL,
    revenue_usd DECIMAL(15, 2)
);

CREATE INDEX idx_orders_date ON orders(order_date);
CREATE INDEX idx_orders_revenue ON orders(revenue_usd DESC);
